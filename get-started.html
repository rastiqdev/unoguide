# Sélection d'une méthode

Selon l'année de votre Unowhy Y13, différentes méthodes existent pour effectuer un jailbreak. Veuillez choisir la méthode correspondante avec le tableau ci-dessous.

AnnéeMéthode
2024Déverouillage de l'UEFI (court-circuit/UTPE, Gen 2)
2022-2023Déverouillage de l'UEFI (court-circuit/UTPE, Gen 1)
2019-2021Mots de passe de l'UEFI

# Pourquoi les mots de passe de l'UEFI n'ont pas été trouvés pour les Y13 construits après 2021 ?

Les Unowhy Y13 ont comme UEFI (quelque chose qui permet de démarrer un système d'exploitation et intéragir avec, voir iciOuvrir dans une nouvelle fenêtre pour plus de détails) AMI AptioOuvrir dans une nouvelle fenêtre. Une des grandes vulnérabilités de cet UEFI est qu'il est possible de retrouver les mots de passe contenus dedans.

Normalement, ces mots de passe sont protégés dans une variable de l'UEFI appelée AMITSESetup, en étant chiffré avec un algorithme de chiffrement XOROuvrir dans une nouvelle fenêtre, sauf que des gens ont trouvé la clé XOR pour les déchiffrer. Tous les mots de passe de l'UEFI sur les Y13 de 2019-2021 ont donc fuité.

Suite à ça, Unowhy a personnalisé AMI Aptio pour utiliser un système de sécurité plus robuste, un système de salageOuvrir dans une nouvelle fenêtre + hachageOuvrir dans une nouvelle fenêtre, ce qui rend les mots de passe presque impossible à déchiffrer.
